# SistemaWeb

## Technologias Utilizadas

-   Reactjs
-   TypeScript
-   Chackra UI
-   React Query
-   React-Router-Dom
-   React-Toastify
-   React-Webcam
-   Zustand

## Requisitos

-   Node versão >= 20

## Instalação

-   Clone o repositório para sua máquina local com um sistema Linux Ubuntu/Mint ou WSL
-   Instale as dependências e rode o projeto com " npm start "
